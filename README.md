# 私人健身与教练预约管理系统 java874

## 项目概述

私人健身与教练预约管理系统是一款基于B/S架构的应用，致力于提供便捷的在线健身预约服务。该系统支持前后端分离，便于维护和扩展，适合作为毕业设计项目。

## 技术栈

- 后端：Java，SpringBoot
- 前端：Vue
- 数据库：MySQL 5.7
- JDK版本：1.8

## 功能模块

系统包含以下主要功能模块：

- 用户模块
- 教练模块
- 管理员模块

各模块具体功能如下：

### 用户模块

- 预约教练
- 查看预约记录
- 修改个人信息

### 教练模块

- 管理预约
- 发布课程
- 修改个人资料

### 管理员模块

- 用户管理
- 教练管理
- 预约管理

## 系统角色

- 用户
- 教练
- 管理员

## 运行环境

- 操作系统：Windows
- 开发环境：IDEA
